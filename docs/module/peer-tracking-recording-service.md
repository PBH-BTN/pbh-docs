# Peer 跟踪记录服务

本服务通过记录 Peer 的连接会话信息、数据传输信息、状态信息等，为需要此功能的模块提供数据支持。

禁用此功能将停止数据收集和处理，依赖该服务的模块可能停止工作，并报告错误数据或空数据。

目前需要此服务的模块有：

* BTN Legacy Protocol (BTNv1 Protocol) 跨种群在线分析
* 状态 -> 当前状态 -> 恶意:正常 连接比例
* 状态 -> 当前状态 -> Peer 会话跟踪数量
* 数据透视 -> 种子 -> 访问历史
* 数据透视 -> IP 查询 -> 访问历史
* 统计 -> 图表 -> Peer ID
* 统计 -> 图表 -> 位置与运营商
* 统计 -> 图表 -> 趋势
